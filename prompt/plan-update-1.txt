# FreeWheel Ad Validation Framework - Implementation Plan

## Status: Phase 1-3 IMPLEMENTED ✓

---

## Architecture

```
packages/validation/     - Pure validation library (model, state machine, rules, report)
packages/freewheel/      - Runtime collector hooks into existing SDK event system
  src/runtime/           - AdRuntimeCollector (new)
apps/demo/               - window.__AD_RUNTIME__ exposure (opt-in via env var)
apps/e2e/                - Playwright E2E test suite
```

---

## Phase 1: Runtime Ad Model & State Machine (DONE ✓)

### packages/validation/
- `src/model.ts` — Unified types: AdSession, AdPod, AdBreak, AdSlotRecord, Creative, Companion, TrackingEvent, TimelineEntry, NetworkRequest
- `src/state-machine.ts` — Per-creative lifecycle FSM: NOT_REQUESTED → REQUESTED → RESPONSE_RECEIVED → SCHEDULED → RENDERED → VISIBLE → STARTED → Q1 → Q2 → Q3 → COMPLETE (with SKIPPED/ERROR terminals)
- `src/state-machine.spec.ts` — 9 unit tests (all passing)
- `src/rules.ts` — 9 validation rules: preroll-before-content, midroll-at-correct-time, postroll-after-content, overlay-during-content, pause-ad-during-pause, all-ads-played, no-duplicate-impressions, creatives-complete, tracking-events-fired
- `src/network.ts` — FreeWheel URL parser, network correlation engine
- `src/report.ts` — JSON + HTML report generation with coverage matrix
- `src/index.ts` — Public API

## Running

```bash
# Unit tests
pnpm --filter @hyoga-fp/validation test

# Full build
pnpm run build

# E2E tests (with demo running on localhost:5173 + VITE_ENABLE_AD_RUNTIME=true)
cd apps/e2e && npx playwright test

# Or in CI mode (auto-starts demo server)
CI=true pnpm --filter @hyoga-fp/e2e test
```

---

## Decisions Made

- validation package has NO dependency on freewheel (avoids circular dep)
- freewheel depends on validation for types
- Collector opt-in via env var (zero production impact)
- Quartile skipping allowed in state machine (common with short ads)
- Self-configuring tests derive all expectations from runtime state (no hardcoded ad IDs)
- Playwright chosen over Puppeteer (better CI support, trace viewer)
